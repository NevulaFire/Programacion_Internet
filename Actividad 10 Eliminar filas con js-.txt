Programar una función para eliminar filas <tr></tr> con javascript
Usar y adaptar el código de la Actividad 03, ver archivos adjuntos como referencia.
Se debe de utilizar JQUERY, se adjunta la librería

Requerimientos:
1. Crear un documento con la estructura básica de un HTML vista en clase
2. Crear la tabla de acuerdo al archivo adjunto (3 columnas, 1 fila para el título de la tabla y 5 filas para las imágenes)
3. Agregar 5 imágenes <img> que sean links <a></a> en la primera columna y su nombre (también link) en la segunda columna
4. Cada link (imágenes y nombres) debe de abrir en una nueva pestaña el enlace correspondiente
5. En la tercera columna agregar un link o un botón que mande llamar una función para eliminar (ocultar) la fila correspondiente con JQUERY
6. Este botón o link debe de preguntar si se quiere eliminar la fila, para esto se debe de usar la función confirm() de javascript
7. Se debe de resolver con UNA SOLA función

Entregables:
1. Subir un archivo (video) con el funcionamiento, debiendo incluir las siguientes pruebas:
a) Hacer click al botón o link de "Eliminar" para ver su funcionamiento, repetir esto tres veces
b) Refrescar navegador
2. Subir los archivos de la practica que contenga el HTML y la función de javascript
